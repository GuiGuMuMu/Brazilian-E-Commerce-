# Data Analytics

This folder contains all data analytics work for the Brazilian E-Commerce project.

## Structure

```
data-analytics/
├── notebooks/       # Jupyter notebooks for exploration and analysis
├── data/
│   ├── raw/         # Original, unmodified data files
│   └── processed/   # Cleaned and transformed data
└── outputs/
    ├── charts/      # Generated visualizations
    └── reports/     # Final reports and summaries
```

## Getting Started

1. Install dependencies: `pip install pandas matplotlib seaborn jupyter`
2. Launch Jupyter: `jupyter notebook`
3. Open any notebook inside `notebooks/`
